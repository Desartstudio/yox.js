yoxjs

Classic Theme

This is the first official yox.js theme. It introduces some core modules: data, view, thumbnails and scroll.

Features

  • Fullscreen support (on browsers that don't support fullscreen, the viewer covers the whole window).
  • Thumbnails with scrolling, gestures and paging.
  • Contents always fit the available space.
  • Contents can be loaded from files, Picasa, Flickr, YouTube, HTML elements or plain HTML.
  • Keyboard-enabled.

Javascript jQuery 

    var yoxApi = new Yox(document.getElementById("yox"), {
        theme: "classic",
        showCopyright: true,
        data: {
            type: "picasa",
            cache: true,
            source: {
                url: "https://picasaweb.google.com/114231305708772868829/PhotographyPortfolio",
                cropThumbnails: true
            }
        }
    });
